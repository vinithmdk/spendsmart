Firebase Setup
One-time setup by admin only
1
Create Firebase Project
Go to console.firebase.google.com → Add project → Name it "SpendSmart" → Continue (disable Analytics) → Create
2
Enable Realtime Database
In Firebase Console → Build → Realtime Database → Create Database → Start in test mode → Enable
3
Get Config Keys
Project Settings (⚙️) → General → Your apps → Add app → Web (</>) → Register → Copy the firebaseConfig object values below
